# 功能特性

## 已支持的功能

### 单元格操作
- 单元格编辑（内嵌编辑器）
- 复制、剪切、粘贴
- 撤销 / 重做
- 合并单元格
- 自动填充

### 样式设置
- 字体、字号、颜色
- 加粗、斜体、下划线、删除线
- 背景色
- 文字对齐（水平和垂直）
- 文字换行
- 单元格边框（支持自定义样式）

### 行列操作
- 添加 / 删除行列
- 隐藏 / 显示行列
- 调整行高和列宽
- 冻结行列

### 多表支持
- 添加 / 删除表格
- 重命名表格
- 切换表格

### 数据操作
- 从 JSON 加载数据
- 导出数据为 JSON
- 数据验证
- 自动筛选

### 界面组件
- 可配置工具栏
- 右键菜单
- 底部表格标签栏
- 滚动条
